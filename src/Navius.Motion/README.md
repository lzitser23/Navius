# Navius.Motion

Standalone Blazor motion primitives and generated CSS for springs, presence, gestures, auto-animation, height animation, and small interaction effects.

## Install

```bash
dotnet add package Navius.Motion --prerelease
```

Reference the package stylesheet from your app shell:

```html
<link href="_content/Navius.Motion/navius-motion.css" rel="stylesheet" />
```

The stylesheet is generated by the C# motion CSS generator and checked into the package as a static web asset.
